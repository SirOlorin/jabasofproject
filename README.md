# jabasofproject
APP Informatique G3B 2018 2019

LE MODELE MVC N'EST PAS ENCORE MIS EN PLACE

Nom des fichiers :
nom de la page = page vitrine
bo + nom de la page = page back office
verbe + nom de la page = action sur la page

exemple :
acceuil.php est la page vitrine visible par tous
boaccueil.php est la page de modification accessible par l'administrateur
addacceuil.php est la page qui permet d'ajouter du contenu à l'accueil
editaccueil.php est la page qui permet de modifier un contenu selectionné dans boaccueil.php
deleteaccueil.php est la page qui permet de supprimer un contenu selectionné dans boaccueil.php
accueil.css
L'ensemble des fichiers d'une page sont reliés par le même fichier css nom-du-fichier.css

/!\ Pour les fichiers annexes (photos ou autre), ne pas utiliser des accès en chemin local, utiliser le dossier racine "jabasofproject" ou une URL.
